# Customer Segmentation & RFM

## Problem statement

You are provided with a dataset of transactions/users/orders. Use SQL to answer business questions.

## Suggested SQL questions

1. Compute RFM (Recency, Frequency, Monetary) for each customer.
2. Rank customers by monetary value using window functions.
3. Create 4 groups using SQL logic (High, Mid, Low, New).
4. Monthly cohort retention rates.
5. Top categories per segment.
6. Average lifetime value per cohort.
7. Customers with increasing spend trend over 6 months.
